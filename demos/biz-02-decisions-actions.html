How this was made — the prompt, where it runs, and an honesty note

The prompt

From this meeting transcript [paste], build a single self-contained HTML
file with two tables — decisions (id, decision, rationale) and actions
(owner, due date, status) — filterable by owner, with checkboxes to mark
actions done and a button that copies the open actions as a Markdown list
I can paste into Teams. Inline CSS/JS only, one file.

Where this runs

Microsoft 365 Copilot Pages lets you create, edit, and preview runnable code directly inside a page (“Build lightweight apps within Microsoft 365 Copilot Pages”, Microsoft Support). Code previews in Copilot Chat and Pages are governed by a Cloud Policy that is enabled by default and can be disabled by your admin (Microsoft Learn: admin configuration). Universal fallback that needs no special features: ask Copilot for a single HTML file, copy the code into a file saved as .html, and open it in your browser.

Honesty note

This is a reference artifact, built with an AI coding agent (Claude Code) to illustrate the output format — it was not generated by M365 Copilot. The prompt above is what you would give Copilot to produce something like it. Results vary by model, tenant configuration, and licence: treat this as a pattern, not a guaranteed output.
